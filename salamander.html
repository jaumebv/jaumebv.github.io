Procedural Salamander AI

2D Simulation · Raylib (C++) · 2023/24

Overview

Procedural Salamander AI is a 2D simulation built in C++ using Raylib by a three-person team. Inspired by Rain World and similar titles, it explores how inverse kinematics, procedural animation, and AI-driven behaviours can generate lifelike, emergent movement without using traditional keyframed animation.


  • Engine: Raylib (C++)
  • Team size: 3 developers
  • Scope: AI · Animation
  • Year: 2023/24
  • Project type: Academic (AI module final project)

Key Features

Inverse Kinematics for Limbs

Implemented custom 2D IK algorithms to animate the salamander’s legs without pre-made keyframes. Each limb dynamically adjusts to maintain natural poses and avoid intersections, resulting in smooth, grounded movement.

Procedural Tail and Body Motion

Developed a spring-mass physics system to simulate the salamander’s flexible tail and overall body dynamics. This system gives the creature a fluid, snake-like feel that reacts organically to movement and collisions.

Reactive Fly Behaviour

Implemented simple AI agents for flies that wander randomly and flee when the salamander approaches. Combined with the salamander’s field-of-view “kill zone”, this creates emergent predator-prey interactions that feel spontaneous.
