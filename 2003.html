2003

A minimalist puzzle game for iPhone, focused on smooth touch feedback and elegant motion.

Overview

2003 is a minimalist tile-based puzzle game developed in Unity as a class project, designed for release on the App Store for iPhone using Xcode and Apple’s publishing tools. The game was created by me, Juan Avilés, and Nacho Avilés. It is built primarily around Unity’s UI interface system, featuring polished transitions, subtle menu animations, and a strong attention to visual detail. The result is a smooth and cohesive experience that feels carefully crafted from start to finish.

Polish & Details

Subtle Guidance

From the very start, the player is subtly guided through the controls with implicit tutorials that rely on motion and visual cues rather than text. Every movement is designed to feel intuitive, helping players understand the mechanics naturally.

Smooth Transitions

Great attention was given to animation and timing, transitions between menus and gameplay are smooth and consistent, creating a cohesive and polished flow throughout the experience.

Accessibility & Customisation

The game also includes basic accessibility features and settings such as sound and music control, and the option to enable or disable screen shake. These small touches make the experience more comfortable for all players.
